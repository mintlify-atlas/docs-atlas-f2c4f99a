---
title: "Introduction"
description: "Expect is a CLI tool that lets AI agents test your code changes in a real browser — one command from scan to pass/fail."
---

Expect reads your unstaged changes or branch diff, sends them to an AI agent (Claude Code or Codex CLI), and generates a step-by-step adversarial test plan. You review the plan in an interactive TUI, then the agent executes each step against a live Playwright browser with your real login sessions injected automatically. Every session is recorded so you can replay exactly what happened.

## Who it's for

Expect is built for developers who use terminal coding agents — **Claude Code**, **Codex CLI**, or **Cursor** — and want automated browser-based validation of their changes without writing or maintaining test scripts.

## Key capabilities

  <Card title="Scan changes" icon="code-branch" href="/how-it-works">

## Prerequisites

Before installing Expect, make sure you have:

- **Node.js 18 or later** — required by `expect-cli`
- **One of the following coding agents installed:**
  - [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
  - [Codex CLI](https://github.com/openai/codex)
  - [Cursor](https://cursor.com)

<Note>
  Expect detects which agent is available and uses it automatically. If none are found, `expect-cli init` will show you installation links.
</Note>
