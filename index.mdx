Expect is a CLI tool that bridges the gap between writing code and verifying it works. Point it at your unstaged changes or branch diff — it reads the code, generates an adversarial test plan using Claude Code or Codex, and executes each step against a real Playwright browser with your actual login sessions injected automatically.

## How Expect works

<Steps>
  <Step title="Scan your changes">
    Expect reads your unstaged edits or branch diff and builds a precise picture of what changed — which files, which lines, which UI surfaces are affected.
  </Step>
  <Step title="Generate a test plan">
    An AI agent (Claude Code or Codex) reads the diff and writes a step-by-step adversarial test plan: not just "check it renders" but "submit the form empty, with bad data, and with valid credentials."
  </Step>
  <Step title="Review and approve">
    The interactive TUI shows you the plan. Edit steps, adjust the instruction, or press `y` to run immediately. Pass `-y` to skip review entirely in CI.
  </Step>
  <Step title="Execute in a real browser">
    Playwright runs each step against your live dev server. Your browser cookies are injected automatically — no manual login required. Every session is recorded for replay.
  </Step>
</Steps>
